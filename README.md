# 项目简介

## 核心功能
- 基于 YOLOv8 农作物检测系统，支持11类农作物识别（蓝草、藜、刺菜、玉米、莎草、棉花、茄属植物、番茄、天鹅绒、生菜、萝卜）
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供模型选择功能和自定义参数设置（置信度、IOU阈值）
- 实时显示检测结果：目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有结果
- 提供检测结果保存功能

## 适用场景
- 农业科研与作物识别
- 农作物监测与分类
- 农业自动化检测
- 农作物病虫害早期识别

## 技术特点
- 使用 PyCharm+Anaconda 或 VSCode+Anaconda 开发环境
- 技术栈：Python3.9、OpenCV-Python、PyQt5、Ultralytics、Torch1.9
- 包含完整源码、数据集（2700张CWC数据集）、训练好的模型及结果
- 提供详细运行教程和UI界面源文件
- 已配置好环境，可直接使用
